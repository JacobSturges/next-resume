# Next Resume

Jacob Sturges web resume built in NextJS

## Getting Started

First install node mopdules, then run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Linting

```bash
npm run lint
# or
yarn lint
# or
pnpm lint
# or
bun lint
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
